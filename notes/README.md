# Historical Backgammon Notes

Files in this directory are immutable historical context. They record what was
believed at the time, including failed investigations and conclusions based on
artifacts that are now quarantined.

Do not copy conclusions or numerical results from these notes into current
plans, dashboards, release notes, or model cards. The authoritative status and
revalidation queue are in [`docs/backgammon_status.md`](../docs/backgammon_status.md).
